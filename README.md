# 🌿 Zen Shapes

A calm, minimal browser game about presence over performance. No timers. No pressure. Just shapes drifting in and out of a warm, earthy screen — and the quiet satisfaction of noticing them.

---

## What is it?

Zen Shapes is a single-file HTML game designed for mindful interaction. You choose a shape, then watch as matching and opposing shapes slowly fade in and out across the screen. Tap them as they float by, or simply watch. When you're ready to surface, a gentle prompt brings you back.

There's also a **Zen Screen** mode — a pure screensaver with no interaction, just all six shapes breathing in and out. Good for a moment of stillness.

---

## How to play

1. A calming quote greets you on launch
2. Read the short instructions, then choose your shape
3. Matching shapes appear in **black** — opposing shapes in **brown**
4. Touch any shape as it drifts by — it glows and dissolves
5. When you're done, touch **"tap to wake"** to see your session score
6. Or tap **zen screen** for a no-interaction screensaver mode

---

## Features

- 🌑 6 shapes: circle, triangle, square, pentagon, hexagon, star
- 🤎 Earthy palette — warm tan background, black and brown outlines
- ✨ Glow-on-touch feedback with soft dissolve animation
- 💬 10 randomised calming quotes on launch
- 📊 Session score (matched vs opposed shapes tapped)
- 🌿 Zen Screen mode — pure screensaver, no interaction
- 📱 Mobile-first, works in any browser
- 0️⃣ No dependencies, no frameworks — single HTML file

---

## Running it

No installation needed. Just open `zen-shapes.html` in any modern browser.

```bash
# Clone the repo
git clone https://github.com/toniqh/zen-shapes.git

# Open the file
open zen-shapes.html
```

Or drag and drop `zen-shapes.html` into your browser window.
